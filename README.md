# Middle Earth Geography
A website that shows the evolution of the landscape of Middle Earth througth the Ages of Arda.

## How to Run
  - Download the project from Github
  - Run the project through an IDE
  - Install Angular with **npm install -g @angular/cli**
  - Run the project using the command **ng serve**

## Current Maps
This project is being updated and so not all maps will be accurate at all times.
